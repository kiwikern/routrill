Routrill was developed to participate in a competition hosted by IT-Talents. It implements the Traveling salesperson problem using four heuristics and let's you compare the different solutions.

It uses Angular 2 for the framework and the Google Maps Javascript APIs for autocompletion, distance calculation and maps. Due to the limitations of their free api plan, only up to 10 destinations can be entered.
